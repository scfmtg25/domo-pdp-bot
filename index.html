Domo API Credentials
To get your Client ID and Secret: In Domo, go to Admin → Developer → New Client. Give it a name, select Data scope, and copy the credentials here. Your instance name is the part before .domo.com in your URL.
e.g. mycompany (from mycompany.domo.com)
Dataset IDs
Find each Dataset ID by opening the dataset in Domo — the ID is in the URL:
domo.com/datasources/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/details/...
User + Branch Data
Paste your user list as CSV. Each row = one user. The branch number names the PDP rule (Branch XXXX) and filters on Full Branch Name.
email,branch_number
john.doe@company.com,21300
jane.smith@company.com,21300
bob.jones@company.com,44500
Include the header row. Columns: email, branch_number
